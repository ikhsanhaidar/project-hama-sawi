PestGuard adalah sistem deteksi hama pada tanaman sawi menggunakan model CNN. Sistem ini menggunakan empat model berbeda untuk mendeteksi apakah gambar yang diinput mengandung hama atau tidak.

## Fitur

1. **Dashboard**: Menampilkan performa dari empat model CNN.
2. **Image Analysis**: Halaman untuk mengunggah gambar dan mendapatkan prediksi dari keempat model.
3. **Model yang Digunakan**:
   - CNN from Scratch (Model 1)
   - VGG16 Transfer Learning (Model 2)
   - Xception Model (Model 3)
   - NASNetMobile (Model 4)
